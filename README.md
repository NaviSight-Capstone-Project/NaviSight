# NaviSight
This is a capstone project that aims to support visually impaired users (VIUs) and their caregivers. This project is also made to fulfill the requirements in graduating in the course of Bachelor of Science in Information Technology (BSIT). 

This project is made mainly by three people but this project could and most likely will not be completed without the appreciated external support and guidance.

## Installation Requirements (.properties)
The following must be placed in app/src/main/assets directory:
* cloudinary.properties
* google-services.json
* smtp.properties
* webrtc-credentials.properties

## Main Features
- [x] WebRTC Audio and Video Call
- [x] QR Pairing
- [x] Location Tracking
- [x] Object Detection
- [ ] Emergency System
