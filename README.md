# NaviSight
This is a capstone project to fulfill the requirements in graduating in the course of Bachelor of Science in Information Technology. This project is made mainly by three people but the project could not be fully completed without the appreciated external support and guidance.

## Installation Requirements (.properties)
The following must be placed in app/src/main/assets directory:
* cloudinary.properties
* google-services.json
* smtp.properties
* webrtc-credentials.properties
